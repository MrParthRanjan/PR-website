## How to Update

### Option 1: Using Find & Replace in VS Code
1. Open `public/index.html`
2. Press `Ctrl+H` (or `Cmd+H` on Mac)
3. Find & Replace each placeholder with your actual URL
4. Save the file (live-reload will trigger)

### Option 2: Direct Edit
1. Open `public/index.html` in VS Code
2. Search for the placeholder URLs (e.g., `https://t.me`, `https://wa.me`, etc.)
3. Replace them with your actual links
4. Save file

The changes will automatically reload in your browser thanks to BrowserSync!
